# Claude Code Getting Started Guide

A single-page HTML guide for non-technical teammates to get up and running with Claude Code in the terminal.

## What's in it

1. **What is Claude Code?** — Simple and proper definitions, explains LLM → Agent → Harness
2. **Get Claude Code Running** — Terminal basics, installation, three launch modes
3. **Level Up Your Terminal** — iTerm2 + Homebrew + Starship + shell config (one-paste setup commands)
4. **Customize Claude Code** — Status bar with context usage, git branch, model info
5. **Add Superpowers with Skills** — Humanizer, frontend-design, and the Skills Marketplace
6. **Recommended Tools** — Wispr Flow (voice dictation) and Raycast (window management, clipboard history, snippets, quicklinks, file search)

## Usage

Open `index.html` in a browser. No build step, no dependencies, no server needed.

```
open index.html
```
